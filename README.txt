Instructions for assignment 4

All the files can be found here: https://github.com/ismita/reinforcement_learning_cs7641

The files
hw4.py and mdp.py contains the code for this assignment and requirements.txt lists the required packages necessary. Put them both under the same directory and run hw4.py
Experiments are done in Python 3.7 environment.

The installation
Before running the .py file, open up a terminal or a command prompt and navigate to the directory of this Python project and type the following command:

pip install -r requirements.txt

This will install all the required python modules for this assignment.
